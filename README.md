# nts-08__-goddhqj8000
엔테크 서비스 SW 개발 - 사전 과제

# 기술 스택
- React, Typescript
- NodeJS, Express
- MySQL
- Heroku

# 프로젝트 이용방법

```
  1. git clone 
  2. npm run install
  3. 로컬 mySQL 환경에 맞추어 /server/config/congig.json 설정
  4. 로컬 mySQL에서 DDL 파일 실행시켜 DB 초기화
  - mysql -u root -p < server/schema.sql
  
  5. npm run start:server
  6. npm run start:client
```
